# hw3 - Hello World Test App

A starter project for building and running a Hello World application.

## Prerequisites

Before you begin, ensure the following tools are installed:

- [Git](https://git-scm.com/) (v2.30+)
- [Node.js](https://nodejs.org/) (v18+ recommended)
- [npm](https://www.npmjs.com/) (included with Node.js)

## Setup

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd hw3
   ```

2. **Install dependencies** (once a `package.json` is added):

   ```bash
   npm install
   ```

## Running the Project

Once application code and dependencies are in place:

```bash
# Start in development mode
npm run dev

# Or build and run the production version
npm run build
npm start
```

> **Note:** This project is in its initial state. The commands above will become functional as source code and configuration files are added.
